### 模块一：产品概况与定位 (The Identity)

*   **产品 Slogan:** “你的私有知识地下城：把吃灰文档变成多巴胺提款机。”
*   **核心一句话定位:** 一款内容无关的 UGC 游戏化知识引擎，通过 AI 地下城主将你上传的任何枯燥 PDF 和笔记，自动转化为防作弊、易上瘾的每日多巴胺通关挑战。
*   **核心价值主张 (Value Props):**
    *   **针对“职场冲刺者” (高付费意愿)：** 别再被动划线 50 页的合规文档与行业财报了！把它们扔进引擎，AI 会自动将其转化为限时扫雷战。在每天通勤的 10 分钟内用爽快连击打通知识盲区，用赚取的通关代币直接解锁高阶 AI 深度研报（把 AI 支出转化为学习收益）。
    *   **针对“备考刷题党” (高活跃度)：** 终结 Anki 的“制卡瘫痪”与“断签焦虑”！一键导入 USMLE 或法考教纲，化身排位赛玩家。遭遇复习地狱？用赚来的代币购买“连胜激冻 (Streak Freeze)”为你的 120 天打卡记录续命，用无情的游戏化收割你的备考多巴胺。

### 模块二：核心功能与用户体验流 (Core Features & UX Flow)

我们不卖“总结”，我们卖“认知摩擦”与“纪律剧场”。以下是用户单次多巴胺循环（User Flow）的具体业务流：

#### 1. 魔法熔炉 (The Forge) - 内容解析与游戏化生成
用户拖拽任意 PDF 或网页链接进入系统。底层 AI 严格遵循“最小信息原则 (Minimum information principle)”进行全自动拆解。为了制造具体的认知摩擦，系统不给长篇大论的总结，而是生成 3 种极具视觉反馈的闯关模式：
*   **模式 A：真伪生存战 (Swipe Survival) - 破冰极速流**
    *   **交互：** 类似 Tinder 的左滑右滑。AI 从文档中提取核心知识点，生成真/伪陈述句。用户必须在 3 秒内心智判断并滑动。连续滑对 5 次触发屏幕震动和暴击音效（Combo Multiplier），激发短期多巴胺。
*   **模式 B：知识盲盒连击 (Cloze Combo) - 间隔重复流**
    *   **交互：** 将核心概念挖空（Cloze Deletion）。界面呈现类似音游的掉落块，用户需要在倒计时结束前点击正确的概念填入坑位。填补 Anki 缺乏即时反馈的弱点，答对直接掉落金币（代币）的视觉动效。
*   **模式 C：AI 领主拷问 (Dungeon Master Interrogation) - 深度主动回忆**
    *   **交互：** 由 AI 化身地下城主（DM），基于用户上传的复杂逻辑抛出一个刁钻的开放式提问。用户必须输入或语音说出关键“咒语（核心知识点）”。AI 实时校验语义，判定你是否格挡成功。

#### 2. PoCW 防作弊引擎 (风控系统)
我们允许用户用“学习代币”换取真实的“大模型 API 额度”，这极易引来羊毛党刷破产。必须部署“认知工作量证明（PoCW）”：
*   **语义复杂度前置校验：** 若用户上传“123123”或复制粘贴垃圾文本，预筛阶段的 L3 语义分析直接将其质量评分降级，该文档的代币产出乘数强制设为 0（Garbage In, Token Out 熔断）。
*   **超人类反作弊时间锁 (Superhuman Speed Trap)：** 如果用户使用点击脚本，在 3 秒内连续答对 10 道逻辑题（远超人类阅读极值），系统不仅不发代币，还会触发“橙色冻结”甚至“红色封禁”没收奖励，锁定升阶。

#### 3. 连胜庇护所 (The Streak Freeze Shop) - 核心通缩黑洞
这是我们保护 API 现金流的终极防御。真实的 AI API 调用会消耗独立开发者的真金白银（$0.03/次），而“打卡冻结”虚拟道具的边际成本为零。
*   **交互设计：** 在应用最显眼处挂着用户的“燃烧连胜天数”。商城内，AI 高级对话额度标价极高，而“打卡冻结”道具包装成华丽的“免死金牌”。
*   **经济算计：** 当备考党面临因意外要断掉 80 天连胜的心理崩溃时，他们会毫不犹豫地将辛苦赚来的数千代币全部砸向购买“打卡冻结”。用零成本的虚荣心去吸收代币流动性，完美实现经济系统的无损回收。

### 模块四：独立开发者 MVP 冲刺指南 (MVP Definition)

你的资源极度有限，API 成本是悬在头顶的剑。第一版必须极度收敛：

**🔴 P0 级别 (必须有，不写代码不能上线，验证核心闭环)：**
1.  **单维内容管道：** 仅支持上传 `.txt` 或 `.md` 纯文本，以及简易 URL 抓取。硬编码一个极其稳定的 LLM Prompt，将文本转为 JSON 格式的 Q&A 数组（节约解析成本）。
2.  **唯一游戏模式：真伪生存战 (Swipe Survival)。** 开发成本最低（左右滑动 UI），多巴胺反馈最快。带简单的屏幕震动和连击计数。
3.  **极简经济闭环：** 积分（经验值）获取，仅能用于在极简商城购买一个道具——“连胜激冻 (Streak Freeze)”。暂不上线 AI 额度兑换（防 API 破产）。
4.  **基础防刷墙：** 硬编码“每题最少停留 2 秒”的限制，快于此速度不计分。

**🟡 P1 级别 (上线后第一批迭代，拉升留存与变现)：**
1.  **AI 领主拷问模式：** 接入大模型生成交互式问答挑战，制造深度认知摩擦。
2.  **高价值代币兑换：** 开放“消耗代币向 AI 提问”功能，并立刻部署“每周兑换上限”与“动态兑换汇率”，测试玩家在零成本（买冻结）与硬成本（买 AI）之间的消耗比例。
3.  **同级排行榜 (Leagues)：** 引入基于本周获得代币数量的 50 人随机分组排行榜，利用社交攀比逼迫备考党和职场人“开卷”。

**🟢 P2 级别 (暂不考虑，未来画饼，绝不要碰的重资产)：**
1.  **NotebookLM 级别的双人 AI 播客生成：** TTS (语音合成) 成本过高，独立开发者在初期无法承担这种重资产生成功能。
2.  **多模态复杂文档解析：** 处理带复杂图表、乱码的 PDF 扫描件（极易造成大模型幻觉和解析崩溃）。
3.  **大地图与重度 3D 虚拟形象：** 违背了轻量级每日打卡的初衷。

> 💡 **CPO 最终叮嘱：** 记住，大厂可以随时抄走你的“AI 出题按钮”，但他们抄不走一个被“损失厌恶”和“连胜执念”死死绑定、为了保住自己 120 天考研连胜记录而疯狂刷题的**社区经济体**。现在，去敲代码吧。
